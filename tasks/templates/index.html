Use the /api/tasks/ endpoint to manage your tasks with methods like GET, POST, PUT, and DELETE.

Each task includes:

  • title (required)
  • description
  • completed (true or false)

Example JSON to create a task:

{
  "title": "Learn Django",
  "description": "Complete DRF tutorial",
  "completed": false
}
